📖 Overview

WS Downloader adalah aplikasi desktop berbasis Python dan Eel yang dirancang untuk mengunduh media dari berbagai platform dengan efisiensi tinggi. Tidak hanya untuk video, aplikasi ini dilengkapi dengan Image / Manga Browser internal yang mampu mencegat (intercept) gambar secara real-time saat kamu browsing.

Dibangun sepenuhnya dengan vibe coding menggunakan AI sebagai partner — proyek ini dimulai dari pertanyaan sederhana: "Bisakah saya membuat downloader sendiri yang sesuai kebutuhan saya?"

✨ Features

  • Video Downloader — Mendukung berbagai situs (YouTube, dll.) dengan pilihan format video/audio menggunakan engine yt-dlp.
  • Instagram Downloader — Mengambil media dari post Instagram dengan sistem fallback multi-cookie untuk menghindari blokir.
  • Manga / Image Browser — Browser internal terintegrasi yang mampu mendeteksi gambar secara otomatis untuk diunduh dalam format PNG asli dengan penomoran urut (001.png, dst).
  • Threads Downloader — (Coming Soon)

🛠 Tech Stack

📋 Requirements

  • Python 3.10+ — Gunakan Python versi 3.10 atau yang lebih baru.
  • FFmpeg — yt-dlp membutuhkan FFmpeg untuk menggabungkan stream video dan audio resolusi tinggi. Download dan pastikan folder bin telah masuk ke System PATH.
  • Browser — Google Chrome atau Microsoft Edge terinstall di sistem untuk me-render UI Eel.
  • Node.js — opsional tapi direkomendasikan.

📦 Installation

  1. Clone Repository
    git clone https://github.com/wahyusd/ws-downloader.git
cd ws-downloader
  2. Setup Virtual Environment
    python -m venv .venv
.venv\Scripts\activate
  3. Install Dependencies
    pip install -r requirements.txt
  4. Jalankan Aplikasi
    Double-click file batch yang sudah disediakan:
    launch-win.bat     ← untuk menjalankan
update.bat         ← untuk update
    Atau via terminal (pastikan .venv aktif):
    python main.py
💡 Untuk mengunduh dari platform yang membutuhkan login (seperti Instagram), letakkan file cookies .txt format Netscape di folder ui/assets/cookies/ dan rename menjadi insta_cookies.txt dan videos_cookies.txt.

🎮 How to Use

Video / Audio Download

Buka tab Video Downloader, paste URL dari YouTube atau platform lain, pilih format yang diinginkan, lalu klik Download.

Video Downloader — Main Interface

Image / Manga Fetcher

  1. Buka Browser
    Buka tab Image/Manga Fetcher, masukkan URL situs target, lalu klik Open Browser.
  2. Scroll & Deteksi
    Saat melakukan scroll di halaman, gambar yang terdeteksi akan diekstrak dan masuk ke antrean di sidebar kanan aplikasi utama.
  3. Download
    Klik Download. File akan disimpan di folder results/images/ dengan format PNG asli dan penomoran urut (001.png, 002.png, dst).
Image/Manga Fetcher — Real-time Image Interception

Menembus Blokir Situs

Situs manga bajakan atau versi 18+ umumnya diblokir oleh ISP. Jika browser menampilkan pesan Connection Reset atau tidak bisa dimuat:

  • Download dan instal Cloudflare WARP (1.1.1.1).
  • Aktifkan WARP di Windows sebelum menekan tombol Open Browser. Ini akan merutekan ulang seluruh trafik melewati blokir ISP secara instan.
