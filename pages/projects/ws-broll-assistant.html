📖 Overview

WS B-Roll Assistant adalah aplikasi desktop yang dirancang untuk mempercepat alur kerja pembuatan konten video. Menggunakan kekuatan AI — baik cloud (OpenAI, Anthropic, Gemini, OpenRouter, Groq, DeepSeek) maupun lokal (Ollama) — aplikasi ini secara otomatis menganalisis transkrip, membagi segmen, memberikan rekomendasi B-Roll, hingga menentukan chapter YouTube dengan musik yang tepat.

Proyek ini lahir dari frustrasi pribadi: proses editing video yang repetitif dan memakan waktu. Pertanyaannya sederhana — "Bisakah AI mengerjakan bagian membosankan dari editing?"

🔒 API Key disimpan dengan enkripsi AES — tidak disimpan dalam plaintext di sistem kamu.

✨ Fitur Utama

  • Multi-Agent AI Support — Terintegrasi dengan GPT-4, Claude 3.5, Gemini 2.0, OpenRouter, Groq, DeepSeek serta dukungan penuh untuk AI Lokal (Ollama) untuk privasi maksimal.
  • High-Precision Transcription — Menggunakan engine Faster-Whisper (CTranslate2) untuk hasil transkripsi audio ke teks (SRT/TXT) yang sangat cepat dan akurat.
  • Smart Segmentation — Secara otomatis membagi video menjadi segmen-segmen logis berdasarkan narasi.
  • Intelligent B-Roll Search — Memberikan rekomendasi visual mulai dari AI Video Prompt, AI Image Prompt, hingga pencarian otomatis di stok media lokal.
  • YouTube Chapters & Music — Menghasilkan chapter YouTube secara instan lengkap dengan rekomendasi musik (keyword online & stok lokal).
  • Professional UI — Antarmuka modern dengan Tailwind CSS, responsif, mendukung drag-and-drop, dan workflow yang intuitif.

🎮 Workflow

  1. Step 1: Transcribe
    Masukkan file video/audio kamu. Tunggu hingga AI selesai membuat file SRT/TXT.
    2. WS B-Roll Assistant — Transcription
  2. Step 2: Segment
    AI akan menganalisis transkrip dan membaginya menjadi beberapa bagian cerita.
  3. Step 3: B-Roll Review
    Untuk setiap segmen, AI memberikan strategi visual (A-Roll vs B-Roll).
    4. WS B-Roll Assistant — Segment Review
  4. Step 4: Generate B-Roll
    Klik "Generate Recommendations" untuk mendapatkan prompt AI atau mencari file di folder stok kamu.
  5. Step 5: Chapters
    Masuk ke menu Chapters untuk membuat daftar isi video YouTube secara otomatis.
    6. WS B-Roll Assistant — YouTube Chapters

🛠 Tech Stack

Backend: Python 3.x, Faster-Whisper (CTranslate2), Requests (REST API), Cryptography (AES)

Frontend: HTML5, Vanilla JS, CSS3, Tailwind CSS

GUI Framework: Eel (Chrome-based)

📋 Requirements

  • Python 3.9 atau lebih tinggi
  • Google Chrome atau Microsoft Edge (untuk menjalankan GUI Eel)
  • FFmpeg — sudah disertakan dalam folder ffmpeg/
  • Ollama (opsional) — untuk menggunakan AI lokal tanpa API key

📦 Installation

  1. Clone Repository
    git clone https://github.com/wahyusd/ws-broll-assistant.git
cd ws-broll-assistant
  2. Setup Virtual Environment
    python -m venv .venv
.venv\Scripts\activate
  3. Install Dependencies
    pip install -r requirements.txt
  4. Jalankan Aplikasi
    Run-Assistant.bat    ← jalankan aplikasi
Build-App.bat        ← bundle menjadi .exe

⚙️ AI Configuration

Aplikasi ini mendukung berbagai provider AI yang bisa dikonfigurasi melalui menu Settings di dalam aplikasi:

  • Cloud: OpenAI, Anthropic, Gemini, OpenRouter, Groq, DeepSeek
  • Local (Ollama): Jalankan Ollama di port 11434 — tidak perlu API key, semua berjalan di komputermu
🔒 API Key disimpan dengan enkripsi AES — tidak disimpan dalam plaintext di sistem kamu.

📁 Output Structure

Hasil kerja aplikasi disimpan di folder output/:

output/
├── transcript.srt / transcript.txt   ← hasil transkripsi
├── segments.md                        ← daftar segmen video
├── review_segments.json               ← analisis visual per segmen
├── broll_recommendations.json         ← rekomendasi B-Roll (akumulatif)
└── chapters.md                        ← daftar chapter YouTube & musik
