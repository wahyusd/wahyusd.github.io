📖 Overview

WS News Center adalah aggregator berita RSS yang dibangun di atas Tauri v2 (Rust) dan Vite. Fokus utamanya adalah kecepatan dan pengalaman membaca yang bersih — tanpa iklan, tanpa gangguan, konten langsung tersedia meski offline.

Proyek ini lahir dari kebutuhan pribadi: saya ingin membaca berita dari banyak sumber di satu tempat, tanpa perlu buka banyak tab browser. Pertanyaannya: "Bisakah saya membangun RSS reader sendiri yang cepat dan ringan?"

Tersedia dalam 3 platform: Web (browser), Desktop (.exe Windows), dan Android (.apk).

WS News Center — How To Add RSS News

✨ Features

  • Universal RSS Fetcher — Mendukung RSS 2.0, Atom, dan fallback rss2json API.
  • CORS Proxy Chain — Auto-fallback melalui corsproxy.io, codetabs, allorigins untuk feed yang bermasalah.
  • Local-First Storage — Data tersimpan di localStorage (web) / SQLite FTS5 (desktop) — baca offline kapanpun.
  • Smart Deduplication — URL normalization + stable djb2 hash mencegah artikel duplikat.
  • Dynamic Categories — Chip kategori otomatis diekstrak dari artikel.
  • Advanced Filtering — Search, category filter, single-date filter.
  • Article Reader — Internal iframe reader dengan Jina AI fetch + DOM cleanup.
  • Resizable Reader — Dual drag handles (kiri & kanan) untuk adjust lebar artikel.
  • Pin Articles — Pin artikel penting, tetap ada saat cleanup.
  • Auto Cleanup — Hapus artikel lama berdasarkan jumlah hari yang dikonfigurasi.
  • Touch Swipe Refresh — Mobile support — swipe down untuk refresh.
  • Dark Theme — Professional zinc dark palette.

🛠 Tech Stack

Frontend: HTML5, Tailwind CSS 3.4, Vanilla JS

Build Tool: Vite 6 + PostCSS + Autoprefixer

Desktop: Tauri v2 (Rust) + SQLite FTS5

🚀 Quick Start

Run via Web (Browser)

Jalankan langsung di browser tanpa perlu Rust atau compilation:

git clone https://github.com/wahyusd/news-center.git
cd news-center
npm install
npm run dev

Aplikasi akan otomatis terbuka di browser pada http://localhost:1420.

🔌 Dynamic Port: Jika port 1420 sudah digunakan, Vite akan otomatis mencari port berikutnya yang tersedia (1421, 1422, dst). Cek terminal untuk port yang aktif.

Available Scripts

Command Description
npm run dev Start Vite dev server
npm run build Build frontend production
npm run preview Preview production build

Build Desktop (.exe)

Prerequisites: Node.js ≥ 20, Rust ≥ 1.75, dan Windows Build Tools (Visual Studio C++).

npm install
npm run tauri build

Output: src/backend/target/release/bundle/msi/ atau nsis/

Build Android (.apk)

Dukungan Android via Tauri v2 (native, bukan Capacitor). Build menggunakan Android Studio atau CLI:

# Prerequisites: Android Studio + NDK + Rust Android targets
npm install
npm run build
npm run tauri android init    # pertama kali saja
npm run tauri android build
📱 Pastikan environment variable ANDROID_HOME dan NDK_HOME sudah diset. Lihat README untuk setup lengkap.

⚙️ Configuration

Aplikasi menggunakan port 1420 untuk dev server (vite.config.js) dengan strictPort: false agar auto-fallback jika port sibuk.

Default RSS feed: Wired (https://www.wired.com/feed/rss). Tambah feed lain via Settings → RSS Feeds → Add.

Tampilan dark theme menggunakan zinc palette professional (#18181b, #27272a, #3f3f46).
